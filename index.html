Generators

Sample Generators

  • int
  • string
  • vector
  • not-empty
  • hash-map
  • return
  • elements
  • one-of

The Process

  1. Macros create test case
  2. Create bookkeeping code
  3. Run cases
  4. Record failures
  5. Shrink failing cases
  6. Report
